# Create Table Data

This function prepares the data for the table.

## Usage

``` r
createTableData(plotData, tableColumns, tableLabels)
```

## Arguments

- plotData:

  A data.table containing the data to be plotted in the table.

- tableColumns:

  A character vector of column names to be included in the table.

- tableLabels:

  A character vector of labels corresponding to `tableColumns`.

## Value

A data.table containing the prepared table data.
