# resets the scale for discrete shapes to ggplot default

resets the scale for discrete shapes to ggplot default

## Usage

``` r
resetDefaultShapeDiscrete(oldShapeValues = NULL)
```

## Arguments

- oldShapeValues:

  shape values to set
