# Get dimensions of plot components

Get dimensions of plot components

## Usage

``` r
getPlotDimensions(plotObject, exportunits, nCol, nRow, nPanel)
```

## Arguments

- plotObject:

  A ggplot object.

- exportunits:

  Units of the exported figure.

- nCol:

  Number of columns for panel plots.

- nRow:

  Number of rows for panel plots.

- nPanel:

  Number of panels for panel plots.

## Value

A list with dimensions of plot components.
