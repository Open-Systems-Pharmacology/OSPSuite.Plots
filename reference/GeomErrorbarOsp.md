# GeomErrorbarOsp

ggproto object for OSP error bars with cap width in mm units. Use
[`geom_errorbar_osp()`](https://www.open-systems-pharmacology.org/OSPSuite.Plots/reference/geom_errorbar_osp.md)
to add this geom to a ggplot.
