# Example Time Profile Data

A data frame containing example pharmacokinetic time profile data used
in the vignette for demonstrating time profile plotting functionality.

## Usage

``` r
data(exampleDataTimeProfile)
```

## Format

A data frame with time-series pharmacokinetic data containing variables
typically used for concentration-time profiles in PBPK modeling.

## Source

Generated for package demonstration purposes.

## References

`vignette("Time Profile Plots", package = "ospsuite.plots")`
