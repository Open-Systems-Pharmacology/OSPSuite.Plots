# Build grid grob for OSP shape

Build grid grob for OSP shape

## Usage

``` r
.ospGrob(name, cx, cy, half, fill, colour, stroke, alpha)
```

## Arguments

- name:

  shape name

- cx:

  center x (grid unit)

- cy:

  center y (grid unit)

- half:

  half-size (grid unit)

- fill:

  fill color

- colour:

  stroke color

- stroke:

  stroke width

- alpha:

  alpha value

## Value

grid grob
