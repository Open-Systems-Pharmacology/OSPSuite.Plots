# Set OSPSuite plots option with a given key and value.

Set OSPSuite plots option with a given key and value.

## Usage

``` r
setOspsuite.plots.option(optionKey, value)
```

## Arguments

- optionKey:

  The key for the option.

- value:

  The value for the option.
