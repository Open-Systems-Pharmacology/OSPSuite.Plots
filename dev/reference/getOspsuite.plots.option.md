# returns an option value for a option defined by the package OSPSuite.plots

returns an option value for a option defined by the package
OSPSuite.plots

## Usage

``` r
getOspsuite.plots.option(optionKey)
```

## Arguments

- optionKey:

  identifier of option

## Value

option value
