# adds a labels by meta data to ggplot object

adds a labels by meta data to ggplot object

## Usage

``` r
addLabels(plotObject, mappedData)
```

## Arguments

- plotObject:

  A `ggplot` object on which to add the labels

- mappedData:

  MappedData object with information of mapped dimensions and units

## Value

updated `ggplot` object
