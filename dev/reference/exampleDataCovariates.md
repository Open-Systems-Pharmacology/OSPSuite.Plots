# Example Covariates Data

A data frame containing example covariate data used in the vignette for
demonstrating histogram plotting functionality with different grouping
variables.

## Usage

``` r
data(exampleDataCovariates)
```

## Format

A data frame with covariate information typically used in population
pharmacokinetic/pharmacodynamic analyses.

## Source

Generated for package demonstration purposes.
