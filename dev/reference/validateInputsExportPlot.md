# Validate inputs for exporting a plot

Validate inputs for exporting a plot

## Usage

``` r
validateInputsExportPlot(plotObject, filepath, filename, width, height, device)
```

## Arguments

- plotObject:

  A ggplot object to be exported.

- filepath:

  A character string specifying the directory to save the plot.

- filename:

  A character string specifying the name of the file (without path).

- width:

  A numeric value specifying the width of the plot.

- height:

  A numeric value specifying the height of the plot.

- device:

  A character with the device to use
