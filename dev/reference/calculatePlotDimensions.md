# Calculate plot dimensions based on the plot object and specified width

Calculate plot dimensions based on the plot object and specified width

## Usage

``` r
calculatePlotDimensions(plotObject, width)
```

## Arguments

- plotObject:

  A ggplot object.

- width:

  A numeric value specifying the width of the plot.

## Value

A list containing the calculated width, height, heightOffset, and
widthOffset of the plot.
