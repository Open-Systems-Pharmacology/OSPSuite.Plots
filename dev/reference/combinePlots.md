# Combine Plot and Table

This function combines the plot and the table into a single output.

## Usage

``` r
combinePlots(plotObject, tableObject, relWidths)
```

## Arguments

- plotObject:

  A ggplot object representing the main plot.

- tableObject:

  A ggplot object representing the table.

- relWidths:

  A numeric vector of length 2 specifying the relative widths of the
  plot and table.

## Value

A combined ggplot object containing both the plot and the table.
